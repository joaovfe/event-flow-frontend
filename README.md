## Descrição

Aplicação front-end desenvolvida com React + Vite + Typescript. 

## Instalação e execução

```bash
# Instalar dependencias
$ npm install

# Desenvolvimento (Watch Mode)
$ npm run dev

# Build
$ npm run build
```
